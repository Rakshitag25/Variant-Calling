# Analysis Scripts

This folder contains core analysis and processing scripts for FASTQ files:

## Files:
- `fastq_preprocessor.py` - Basic FASTQ validation and preprocessing
- `chunk_fastq.py` - Split large FASTQ files into manageable chunks
- `batch_process_chunks.py` - Orchestrate processing of multiple chunks
- `fast_file_analysis.py` - Quick file analysis and statistics

## Usage:
These scripts form the foundation of the FASTQ processing pipeline, handling file chunking, validation, and batch processing operations.
