# Visualization

This folder contains visualization scripts and generated plots:

## Scripts:
- `visualize_trimming.py` - Generate comprehensive trimming summary plots
- `visualize_trimming_comparison.py` - Compare before/after trimming statistics
- `compare_reads.py` - Compare original vs trimmed reads with quality analysis
- `detailed_trim_analysis.py` - Detailed analysis of trimming effects

## Generated Plots:
- `Figure_1.png` - Quality control analysis plots
- `read_examples_comparison.png` - Read comparison visualizations
- `trimming_before_after_comparison.png` - Trimming effect comparisons
- Additional plots in `trimming_results/fastpandtrim/`

## Usage:
These scripts generate comprehensive visualizations of quality control metrics, trimming effects, and read comparisons to assess the preprocessing pipeline performance.
